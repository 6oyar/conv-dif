========================================================================
    Численное решение задачи конвекции-диффузии.
========================================================================
Программа решает стационарное уравнение конвекции-диффузии вида:
-div( k*grad(u) ) + v*grad(u) = f(x), f(x) задаётся в методе source(double x, double y),

нестационарное уравнение вида:
du/dt - div( k*grad(u) ) + v*grad(u) = f(x).

С помощью метода конечных элементов строится система линейных уравнений, которая решается методами:
-BiCGSTAB
-BiCGSTAB + ILUT
-CG
-LDLT-разложение
-LLT-разложение

Для реализации данных методов используется библиотека с открытым кодом Eigen.

Решение выводится в формате .dat, для построения графика рекомендуется использовать приложение TecPlot.
